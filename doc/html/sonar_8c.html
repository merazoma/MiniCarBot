◆ excep_sonar_echo_front()

void excep_sonar_echo_front ( )

前側超音波センサエコー割り込み入力端子の立ち下がりエッジで呼ばれる

割り込みハンドラで、エコーON時間を算出のためにタイマカウントを取得 立ち下がりエッジでのMTU3（トリガ信号生成にも使用）のタイマカウントを取得
トリガ信号の立ち上がりエッジ(MTU3.TCNT: 0)とエコー信号の立ち上がりエッジの 時間間隔が一定なので、立ち下がりエッジでのタイマカウント取得で、エコー信号の パルス幅を実行的に取得できる

注意
IRQ0の割り込み処理であるExcep_IRQ0の内部でコールすること
