自動運転ミニカーロボット(Flounder)

自動運転ミニカーロボット(Flounder)の主に、プログラムについてのドキュメントです。

本ドキュメントは、Doxygenで作成されたもので、 関数等の解説はCソースのコメント等から、Doxygenで自動生成されます。

Flounderの概要

マイコンのポート配置

マイコンはRX220の秋月ボード。

ボードのピン名は、秋月のRX220マイコンボードのマニュアルから。

ポート名と周辺モジュール名はRX220のユーザーズマニュアルから抜粋。
